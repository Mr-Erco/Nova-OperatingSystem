# NovaOS
NovaOS is a lightweight, web‑based desktop environment designed to run inside any modern browser or as a standalone PC app. It provides a clean, minimal, and customizable workspace with apps, windows, widgets, and tools — all built using HTML, CSS, and JavaScript.

NovaOS is not a real operating system. It is a UI simulation built for creativity, productivity, and experimentation.

---

## ✨ Features
- **Desktop Environment**  
  A full windowed interface with draggable apps, a taskbar/dock, and a customizable layout.

- **Built‑in Apps**  
  Includes core apps like Notes, Files, Settings, TerminalX (optional), and more.

- **Window Manager**  
  Supports open/close/minimize/maximize, stacking, and multi‑window layouts.

- **Themes & Customization**  
  Change wallpapers, colors, UI styles, and animations.

- **Fast & Lightweight**  
  Runs entirely in the browser with no backend required.

- **Cross‑Platform**  
  Works on Windows, macOS, Linux, and mobile browsers.

---

## 🖥️ Tech Stack
- **HTML5** for structure  
- **CSS3** for UI, themes, and animations  
- **JavaScript (Vanilla)** for logic, apps, and window management  
- Optional: **Electron/Tauri** for packaging as a PC app

---

## 🚀 Getting Started

### Run in Browser
Just open `index.html` in any modern browser.

### Run as a PC App (Optional)
You can package NovaOS using:
- Electron  
- Tauri  
- NW.js  

This gives you a real `.exe` or `.app` file.
